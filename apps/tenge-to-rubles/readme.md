# Конвертер тенге в рубли для геймеров Steam

Конвертер валют специально для геймеров в России, которые перенесли свои аккаунты Steam в регион Казахстан и покупают игры за казахские тенге.

## О проекте

Этот инструмент помогает российским геймерам определить, сколько нужно положить рублей в кошелёк Steam для покупки игр за казахские тенге, учитывая все комиссии.

### Основные возможности

- Конвертация тенге в рубли по актуальному курсу
- Автоматический расчёт с учётом комиссий:
  - 8% комиссия посредников при пополнении кошелька Steam
  - 100 ₽ для компенсации возможных комиссий самого Steam
- Автоматическое переключение темы "от заката до рассвета" (с 20:00 до 6:00)
- Возможность ручного переключения темы
- Минималистичный, современный дизайн
- Полная адаптивность для мобильных устройств
- Подробные подсказки и инструкции

## Как использовать

1. Введите цену игры в казахских тенге (₸)
2. Нажмите кнопку "Конвертировать"
3. Получите результат - сколько рублей нужно положить на счёт Steam
