# Tenge → Rubles — конвертор для геймеров (для размещения на GitHub Pages)

**Автор:** Владимир Моденов  
**URL в вашем сайте:** `/apps/tenge-to-rubles/`

## Что делает проект
Простая одностраничная веб-страница для быстрой конвертации цены из казахских тенге (KZT) в российские рубли (RUB) с учётом:
- актуального курса (запрашивается в реальном времени у `exchangerate.host`),
- дополнительно +8% для компенсации комиссий посредников,
- дополнительно +100 ₽ для возможных комиссий Steam.

Цель — помочь геймерам, которые покупают игры в регионе Казахстан и пополняют российский кошелёк Steam, понять, сколько ₽ нужно положить на счёт.

## Технические детали
- Чистый HTML / CSS / JavaScript (без сборщиков и зависимостей).
- Курс берётся через публичный endpoint: `https://api.exchangerate.host/latest?base=KZT&symbols=RUB`.
- Результаты курса кешируются в `sessionStorage` на 1 час, чтобы снизить нагрузку на API.
- Автоматическое включение тёмной темы: сначала пытаемся определить координаты по IP (`https://ipapi.co/json/`) и получить время восхода/захода (`https://api.sunrise-sunset.org/json`). Если эти запросы недоступны — применяется запасной режим (тёмная тема с 19:00 до 07:00 по системному времени пользователя).
- Дизайн минималистичный: белый фон, шрифт Roboto, скруглённые элементы и простая анимация при наведении.

## Файлы
- `index.html` — главная страница.
- `styles.css` — минималистичный стиль.
- `app.js` — логика (получение курса, расчёты, авто-тема).
- `README.md` — этот файл.

## Как развернуть
1. Скопируйте файлы в папку репозитория GitHub Pages: `mods/.../apps/tenge-to-rubles/` (либо в `gh-pages` ветку).  
2. Убедитесь, что путь совпадает с мета-тегом `og:url` (в `index.html`) при необходимости - поправьте.  
3. Коммитните и запушьте — страница будет доступна по `https://<ваше_имя>.github.io/apps/tenge-to-rubles/`.

## Примечания и рекомендации
- API `exchangerate.host` удобен и бесплатен для большинства простых сценариев, но при интенсивной нагрузке или коммерческом использовании рекомендуем ознакомиться с условиями сервиса и при необходимости получить более надёжный источник или свой API-ключ.
- Поскольку некоторые сторонние API (ipapi, sunrise-sunset) зависят от CORS и внешних сервисов, в оффлайн-режиме автопереключение темы может не сработать — в этом случае будет использоваться простая проверка системного времени (19:00–07:00).
- Если хотите, могу сделать опциональную кнопку «Обновить курс» и/или выбрать ручной ввод курса для офлайн-режима.
