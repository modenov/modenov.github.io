# Конвертер тенге → рубли

Мини-приложение для расчёта стоимости игр в Steam Казахстан в рублях.

## Включает:
- Актуальный курс KZT → RUB через API exchangerate.host.
- 8% надбавка на посредников.
- Дополнительные 100 ₽ на комиссии Steam.
- Автоматическая тёмная тема.
- Минималистичный дизайн.

## Файлы:
- index.html — интерфейс приложения.
- styles.css — оформление.
- app.js — логика.
- README.md — описание проекта.
