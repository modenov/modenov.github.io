# Конвертор тенге → рубли для Steam

**Автор:** Владимир Моденов  
**Назначение:** позволяет геймерам, у которых игры покупаются по курсу KZT, быстро рассчитаться, сколько рублей нужно положить на кошелёк Steam с учётом комиссий.

## Как работает

1. Пользователь вводит цену игры в тенге (KZT).
2. Скрипт запрашивает актуальный курс KZT → RUB через бесплатный API.
3. Результат умножается на **1.08** (плюс 8% — комиссия посредников).
4. Затем добавляется фиксированная надбавка **+100 RUB** (на случай комиссии самого Steam).
5. Показывается итоговая сумма — сколько рублей нужно пополнить.

## Почему это удобно

- Всегда актуальный курс — не нужно вручную смотреть курс обмена.
- Автоматический учёт возможных комиссий и надбавок.
- Минимализм и простота — страница загружается быстро, без лишнего.
- Тёмная тема — удобно для использования ночью.

## Технологии

- HTML + CSS + JavaScript (без фреймворков).
- Бесплатный публичный API: [exchangerate.host](https://api.exchangerate.host) — поддерживает конвертацию KZT → RUB. :contentReference[oaicite:0]{index=0}

## Установка

1. Скопируйте файлы в папку `apps/tenge-to-rubles/` на вашем GitHub Pages.
2. Убедитесь, что `index.html`, `style.css`, `script.js` загружаются.
3. Откройте страницу в браузере — всё готово.

## Настройка

- При необходимости можно скорректировать время для тёмной темы (в `script.js`, функция `applyThemeByTime`).
- Можно поменять надбавку +100 RUB или процент комиссии, если условия изменятся.
- Можно стилизовать иначе — сейчас минимализм + скругления + простые анимации для hover.
